# E-Commerce Product Listing App

This is a simple e-commerce product listing app built using React.
It displays a collection of products with images, names, categories, prices, and descriptions. 
Users can filter products by category using interactive buttons.



Features

- Dynamic product list using React `useState`
- Component-based structure (App to ProductList to ProductItem)
- Props used to pass data between components
- Category filtering with interactive buttons
- Product images 
- Custom CSS styling
